## Overview
Macで動作するPythonのリーディングルーラー（文字読みの補助するやつ）

## Requirement
- MacOS
- Python 3.11.5
- tkinter　6.8

## Usage
pythonプログラムとして実行

## Features
- ウィンドウの大きさをいじると文字を読める部分も変わる
- コード内のカラーコードをいじれば色変えできる
